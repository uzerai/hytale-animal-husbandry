## Introduction
A mod that introduces animal husbandry in the form of growth stages and taming of animals in the early access release of Hytale.

The build setup and directory structure was entirely copied from https://github.com/Build-9/Hytale-Example-Project; a great
resource for any budding modders such as myself.
